# Zoom Account Monitoring System

This project automates the tracking of Zoom account bookings using Google Sheets and Apps Script.  
It includes:
- A Google Form for meeting requests  
- A linked Google Sheet for logging and monitoring schedules  
- Apps Script automation for highlighting conflicts and sending reminders  
- Optional Excel dashboard for visual summaries

## Technologies Used
- Google Sheets  
- Google Apps Script  
- Microsoft Excel  
- HTML, CSS and CSS (for dashboard interface)

## Features
- Automatic conflict detection  
- Color‑coded booking status  
- Clickable dashboard buttons  
- Integration with Zoom account capacity (100 / 300 / 500 pax)
